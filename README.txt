Java program that resolves "Dwarfes and Switches" riddle.
The terms of a riddle : "There are 100 switches and 100 dwarfes,each dwarf has a number on its back 
(from 1  to 100) each dwarf will turn on/off only those swtitches inceremented by the dwarf's number 
so for example dwarf number 3 will change the position of switches 1,3,6,9 and so on...
What switches will change their initial state after all of 100 dwarfes will make their run."
